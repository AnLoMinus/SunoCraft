## 🚀 איך להשתמש בתבנית?

### שלב 1: יצירת מאגר חדש
```bash
# צור תיקייה חדשה
mkdir MySunoProject
cd MySunoProject

# העתק את המבנה
cp -r SC-FullProject-Template/* .
```

### שלב 2: התאמה אישית
1. שנה את `index.html` לפי הצרכים שלך
2. עדכן את `README.md` עם פרטים על הפרויקט שלך
3. הוסף תוכן לתיקיות השונות

### שלב 3: העלאה ל-GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-repo-url>
git push -u origin main
```

## 🎯 דוגמאות שימוש

### דוגמה 1: יצירת תבנית מילים חדשה
1. צור קובץ חדש ב-`lyrics/holy-trap/my-song.md`
2. העתק תבנית מ-`SC-HolyTrap-Lyrics.md`
3. כתוב את המילים שלך
4. שמור וצור שיר ב-SUNO

### דוגמה 2: הוספת פרומפט חדש
1. צור קובץ חדש ב-`prompts/afro-electro/my-prompt.md`
2. כתוב את הפרומפט עם כל הפרטים
3. הוסף דוגמת מילים
4. שמור לשימוש עתידי

### דוגמה 3: יצירת מדריך חדש
1. צור קובץ חדש ב-`guides/my-guide.md`
2. כתוב את המדריך
3. הוסף קישור ב-`index.html`
4. עדכן את `README.md`
