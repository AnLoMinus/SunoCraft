## 📋 מה זה SunoCraft?

**SunoCraft (SC)** הוא מאגר מקיף לכלי SUNO ליצירת מוזיקה קדושה, אלקטרונית וראפ. המאגר כולל:

- 🎵 מדריכים מפורטים ליצירה ב-SUNO
- 📜 פרומפטים מוכנים לשימוש
- 🎤 תבניות מילים ופלואו
- 🎚️ טבלאות BPM וסגנונות
- 📦 תבניות הורדה וכלים

---

## 🚀 התחלה מהירה

### 1. צפייה באתר
פתח את `index.html` בדפדפן כדי לראות את כל המדריכים והמידע.

### 2. הורדת תבניות
כל התבניות זמינות בתיקיית המאגר:
- `SC-HolyTrap-Lyrics.md` - תבנית מילים ל-Trap קדוש
- `SC-AfroElectro-PromptPack.md` - חבילת פרומפטים
- `SC-BPM-Guide.md` - מדריך BPM
- `SC-RapFlow-Workbook.md` - חוברת תרגול פלואו

### 3. יצירת שיר ראשון
1. בחר פרומפט מ-`SC-AfroElectro-PromptPack.md`
2. כתוב מילים לפי `SC-HolyTrap-Lyrics.md`
3. העתק הכל ל-SUNO וצור את השיר!
