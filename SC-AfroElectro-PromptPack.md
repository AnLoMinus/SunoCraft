## 🎧 פרומפט 1: Holy Afro House

### הגדרה:
```
energetic afro house + electro beat, deep kick, wide tribal drums, 
warm sub bass, bright synth sparks, mystical holy vibe, cosmic fire, 
angelic choir pads, shimmering plucks, uplifting emotional energy, 
club ready but spiritual, clean punchy mix.
```

### פרטים:
- **BPM:** 105-118
- **אווירה:** שמחה, שבטי, קדושה רוקדת
- **שימוש:** מסיבות נשמה, חפלות קדושות
- **כלים:** תופים שבטיים, בס עמוק, סינתיסייזרים בהירים, מקהלה

### דוגמת מילים:
```
[Verse]

הגוף זז, הנשמה מתעוררת,
כל פעימה מעלה אותי למעלה,
כל צליל מחבר אותי אליך.

[Chorus]
אני רוקד באור שלך,
אני שר את התפילה,
המוזיקה היא הגשר,
בין השמים לארץ.
```

## 🎯 טיפים לשימוש

### ✅ איך להשתמש בפרומפטים:
1. **העתק את הפרומפט** - בחר את הפרומפט המתאים לך
2. **הוסף מילים** - כתוב את המילים שלך לפי המבנה
3. **התאם לפי הצורך** - שנה פרטים קטנים לפי הרעיון שלך
4. **הרץ ב-SUNO** - העתק הכל ל-SUNO וצור את השיר

### 🔧 התאמות אישיות:
- **שינוי BPM** - התאם את הקצב לפי הצורך
- **הוספת כלים** - הוסף כלים נוספים אם צריך
- **שינוי אווירה** - שנה מילים כמו "mystical" ל-"energetic" או "deep"
- **ערבוב סגנונות** - נסה לשלב בין פרומפטים שונים
